SETUP INSTRUCTIONS
Backend Setup


# Navigate to backend directory
cd backend

# Install dependencies
npm install

# Run database migrations
npm run db:push

# Start backend server
npm run dev




Frontend Setup


# Navigate to frontend directory
cd frontend

# Install dependencies
npm install

# Start frontend development server
npm start



Create First Admin
After both servers are running, create your first admin account:
bashcurl -X POST http://localhost:5000/api/auth/register \
  -H "Content-Type: application/json" \
  -d '{
    "username": "admin",
    "email": "admin@example.com",
    "password": "admin123"
  }'




Access the Application

Frontend: http://localhost:3000
Backend API: http://localhost:5000
Login Credentials:

Username: admin
Password: admin123



IMPORTANT NOTES

Backend runs on port 5000
Frontend runs on port 3000
Database is already configured with your Neon connection string
All files are complete and ready to use
No errors should occur if you follow the setup instructions exactly



Postman (Easy GUI Method)

Download Postman: https://www.postman.com/downloads/
Open Postman
Create a new request:

Method: POST
URL: http://localhost:5000/api/auth/register
Go to Body tab
Select raw and JSON
Paste this:

{
     "username": "admin",
     "email": "admin@example.com",
     "password": "admin123"
   }
